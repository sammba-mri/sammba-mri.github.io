User guide: table of contents

Sammba-MRI is a Python module that provides easy-to-use pipelines to process and analyze MRI multimodal images of small-mammal brain.

Basic MRI preprocessing toolbox relies on several dependencies (AFNI, FSL) that are directly implemented in Sammba-MRI workflow via nipype.
